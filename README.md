# Проект «Оно тебе надо»

Лендинг для аукциона вещей, в которые изначально никто не верил.

## Ссылка на проект

https://github.com/DazhMaks-debug/ono-tebe-nado

## Описание

Верстка адаптивной веб-страницы по макету из Figma. Проект включает:

- Семантическую разметку HTML5
- Адаптивную верстку с использованием Flexbox и Grid
- Кастомные шрифты
- Оптимизированные изображения
- Кроссбраузерную совместимость

## Особенности реализации

1. **Семантическая верстка**:
   - Использованы теги `header`, `main`, `footer`, `section`, `article`, `nav`
   - Корректная иерархия заголовков h1-h6
   - Семантические теги для адреса, списков, параграфов

2. **Адаптивность**:
   - Минимальная ширина страницы 1100px
   - Резиновые элементы с использованием flex и grid
   - Карточки переносятся на новую строку при необходимости

3. **Доступность**:
   - Alt-тексты для всех изображений
   - Семантические ссылки с префиксами tel: и mailto:
   - Правильная фокус-навигация

4. **Оптимизация**:
   - Переиспользование стилей (оверлей, карточки, адрес)
   - Минималистичный CSS
   - Оптимизированные изображения
